<summary>⚙️ <b>Snake nahi dikh raha? Click here for setup</b></summary>

<br/>

**Step 1:** Profile repo mein file banao `.github/workflows/snake.yml`:

```yaml
name: Generate Snake
on:
  schedule: [{ cron: "0 0 * * *" }]
  workflow_dispatch:
jobs:
  snake:
    runs-on: ubuntu-latest
    steps:
      - uses: Platane/snk@v3
        with:
          github_user_name: vinit401
          outputs: |
            dist/github-contribution-grid-snake-dark.svg?palette=github-dark
            dist/github-contribution-grid-snake.svg?palette=github-light
      - uses: crazy-max/ghaction-github-pages@v3.1.0
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

**Step 2:** Actions tab → **Run workflow** manually ek baar.

**Step 3:** Done! Snake auto-update hoga daily. ✅
