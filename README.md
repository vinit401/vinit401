## 🐍 Contribution Snake

> Add this GitHub Action to your profile repo to enable the snake animation!
> File: `.github/workflows/snake.yml`

```yaml
name: Generate Snake
on:
  schedule: [{ cron: "0 0 * * *" }]
  workflow_dispatch:
jobs:
  snake:
    runs-on: ubuntu-latest
    steps:
      - uses: Platane/snk@v3
        with:
          github_user_name: vinit401
          outputs: |
            dist/github-contribution-grid-snake-dark.svg?palette=github-dark
      - uses: crazy-max/ghaction-github-pages@v3.1.0
        with:
          target_branch: output
          build_dir: dist
        env:
          GITHUB_TOKEN: ${{ secrets.GITHUB_TOKEN }}
```

> Then add this line in README to show the snake:
> `![Snake](https://raw.githubusercontent.com/vinit401/vinit401/output/github-contribution-grid-snake-dark.svg)`
